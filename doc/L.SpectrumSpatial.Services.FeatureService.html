 The point used as the starting location for the search. For example: point=-75.651157,45.374245,EPSG:4326

 The distance to search around the point. By default the tolerance is 300 meters

 The geometry definition attribute from the table that should be used for processing the spatial query. This attribute is only required for tables that contain more than one geometry attribute definition.
